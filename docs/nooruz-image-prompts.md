# Nooruz Scene Image Prompts

These prompts are prepared for generating assets via ChatGPT image generation for the Nooruz version of the website.

Important rules for every asset:
- Output as `PNG with transparent background`
- No mockup, no frame, no background scene unless explicitly requested
- No black box, no gradient rectangle, no fake poster background
- Keep one consistent art direction across all assets
- Style target: premium festive web hero, elegant Kyrgyz Nooruz mood, warm sunlight, spring atmosphere, ornamental detail, cinematic but clean

## Recommended Art Direction

Use this style prefix in all prompts:

```text
High-end festive web illustration, Kyrgyz Nooruz celebration, refined ornamental detailing, warm sunrise gold, spring green accents, elegant volumetric glow, premium cinematic lighting, detailed but clean composition, suitable for luxury landing page UI
```

## 1. Full Hero Scene Prompt

Use this if you want one main hero visual instead of separate parts.

```text
High-end festive web illustration, Kyrgyz Nooruz celebration, refined ornamental detailing, warm sunrise gold, spring green accents, elegant volumetric glow, premium cinematic lighting, detailed but clean composition, suitable for luxury landing page UI.

Create a wide Nooruz hero scene for a website landing page. Show a spring sunrise over stylized Central Asian mountains, soft rolling hills, a subtle meadow in the foreground, elegant yurts, a few slender poplar trees, small tulip accents, and a glowing decorative Nooruz centerpiece in the middle. The scene should feel celebratory, calm, premium, and richly layered, with atmospheric depth and soft floating particles. Use the website palette: deep plum night, warm amber-gold sunlight, muted spring greens, copper highlights, parchment glow. Keep the composition balanced and readable for web, with open negative space for UI text on the left. No mosque, no crescent moon, no Ramadan motifs, no lanterns.

Output as PNG, transparent background if possible. If transparent background is not possible for a full scene, keep the background extremely clean and easy to mask. No text unless explicitly added as ornamental centerpiece text. No mockup, no border, no dark rectangle behind the scene.
```

## Best Workflow

1. Generate the `main centerpiece` first.
2. Generate the `landscape wide` layer second.
3. Generate `yurts`, `trees`, and `flowers` as separate transparent PNG parts.
4. Generate `petals/particles` as a separate overlay asset.
5. Keep all prompts in the same style family.

## Practical Tip

If ChatGPT adds an unwanted background, repeat this suffix:

```text
Transparent background required. Isolated object only. No background color, no vignette, no gradient panel, no shadow box, no poster layout.
```
